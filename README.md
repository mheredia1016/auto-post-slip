# Slip Click Helper Bot

Detects betting slip screenshots with OCR and posts a button:

🔥 Tap to generate Gambly link

Because Gambly/Playbook usually ignore other bots, the button gives the real user a private copy/paste instruction.

## Railway Variables

DISCORD_TOKEN=your_bot_token
GAMBLY_MENTION=<@gambly_bot_id>
PLAYBOOK_MENTION=<@playbook_bot_id>
